# EP01: Semantic Kernel이 뭔가요?

Microsoft의 오픈 소스 프로젝트인 Semantic Kernel 프로젝트(시맨틱 커널; SK)를 소개합니다.

- ✅ 시맨틱 커널이 무엇인지
- ✅ 시맨틱 커널이 추구하는 방향은 무엇인지
- ✅ 시맨틱 커널로 할 수 있는 것들
- ✅ 전체 시리즈의 방향

등에 대해 얘기합니다.
